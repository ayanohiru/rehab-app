# スポーツ疾患向け自宅リハビリ支援Webアプリ

## 概要
スポーツ疾患を持つ患者向けに、自宅でリハビリを行えるよう支援するWebアプリ。

## 目的
- 自宅で安全にリハビリを実施できるようにする  
- 理学療法士の処方をわかりやすく提供する  
- 実施状況を可視化する  

## 現在の仕様（要件）
以下の機能を実装する：

- 疾患選択画面  
- 理学療法士による体操の処方表示  
- 動画・図による説明  
- 患者のチェック報告機能  

## 技術要件
- HTML / CSS / JavaScript  
- 1ファイルで完結する（rehab.html）  

## 現在の実装状況
- 疾患選択 → OK  
- 体操表示 → OK  
- 動画表示 → OK  
- 実施チェック → OK
